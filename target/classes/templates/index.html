API ENDPOINTS

POST /api/account/register - Register new user account
POST /api/account/login - User authentication
GET /api/products - Get all products
POST /api/products - Create new product (AUTH REQUIRED)
GET /api/products/{id} - Get product by ID
PUT /api/products/{id} - Update product (AUTH REQUIRED)
PATCH /api/products/{id} - Partial update product (AUTH REQUIRED)
DELETE /api/products/{id} - Delete product (AUTH REQUIRED)
GET /api/orders - Get all orders
POST /api/orders - Create new order (AUTH REQUIRED)
GET /api/orders/{id} - Get order by ID
PUT /api/orders/{id} - Update order (AUTH REQUIRED)
DELETE /api/orders/{id} - Delete order (AUTH REQUIRED)
